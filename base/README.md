## jdbckit
jdbckit 是一个生成简单对象映射的工具（spring jdbc template）

1. 生成简单对象，映射和简单的多条件查询
2. 支持 字段映射成json 和xml
3. 强类型复杂查询

jdbckit 优点

1. 简单
2. 强类型
